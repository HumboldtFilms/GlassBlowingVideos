## 🚀 Overview

This project is built to help glassblowing creators turn their skills into an online business.

It includes:
- High-converting landing page
- SEO-optimized blog section
- Course sales integration
- Content-driven marketing strategy

---

## 🎯 Features

- ⚡ Modern React + Tailwind UI
- 🔍 SEO-optimized structure
- 🎥 Video-focused layout
- 📝 Blog system for organic traffic
- 💰 Built for digital product monetization

---

## 💡 Business Use Case

This platform is designed to:
- Sell glassblowing tutorials
- Generate traffic through SEO
- Convert visitors into paying customers
